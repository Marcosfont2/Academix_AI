# 🚀 Como rodar o Academix AI localmente

Para rodar o projeto completo, você precisará abrir **dois terminais** separados.

### 1. Rodando o Backend (Spring Boot / Java)
Abra o primeiro terminal, navegue até a pasta do backend e execute o comando do Maven:

```bash
cd backend
.\mvnw spring-boot:run

cd frontend
npm run dev
